# ELProject
EL项目
这是南京大学软件学院2014级 easy life easy learn 比赛的参赛作品
# 项目简介

是以《三体》为背景而开发的益智小游戏，要求玩家随意移动地图中的指定星球来控制光线的折射与反射，使得光线到达指定位置的游戏。如下图所示

# 项目亮点
## 地图编辑器

参考 war craft 的辉煌，我们提供了地图编辑器，玩家可以随意编辑自己的地图
![](1487865191462.png)

## 网络交流
提供了网络交流的平台，玩家可以通过互联网分享、获取他人的游戏地图
